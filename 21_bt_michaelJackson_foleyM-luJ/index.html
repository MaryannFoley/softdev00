Containers

With bootstrap, you are able to create containers that will dynamically resize and can be divided into 1-12 sections. This is written as follows:


			<div class="container">
				<div class="row">
					<div class="col-sm">
						The number of times this piece is repeated is the number of columns
					</div>
				</div>
			</div>

Buttons

With bootstrap, you are able to create buttons of many styles without the hastle of making them look nice. This is written as follows:



			<button type="button" class="btn btn-XXXX">Title</button>

The different styles are displayed below with their names (what would replace XXXX in the code above).



Other button styles

Note: These can all be used with all above styles, primary is being used as an example.

Outline buttons

			<button type="button" class="btn btn-outline-primary">Primary</button>



Disabled buttons

			<button type="button" class="btn btn-primary" disabled>Primary</button>


Try pressing the button above!
Note: This does not prevent the button from being clicked at all costs; someone can always edit the html from their browser!



Buttons as a class

			<a class="btn btn-primary" href="#" role="button">Link</a>
			<input class="btn btn-primary" type="submit" value="Submit">
